# Customer.io

<!-- para:0002 -->
Explore customer data and generate insights via Claude

- Used in: [Claude](https://claude.ai/directory/e46d22da-f472-465a-ae46-52f6ac61a97f) [Claude desktop app](https://claude.com/connectors/customer-io#) [Claude mobile app](https://claude.com/connectors/customer-io#) [Claude Code](https://claude.com/connectors/customer-io#) [Claude Code](https://claude.com/connectors/customer-io#) [Claude API](https://claude.com/connectors/customer-io#) [Claude API](https://claude.com/connectors/customer-io#)

<!-- para:0005 -->
Let Claude work directly with your Customer.io workspace to create segments, inspect user profiles, search for customers, and access workspace data. Analyze customer attributes, manage audience targeting, and explore your workspace without switching tabs.

<!-- para:0006 -->
You can use Customer.io to:

<!-- para:0007 -->
**Customer Segmentation: ** "I want to create a segment of users who haven't been active in the last 14 days for a re-engagement campaign. Can you help me set this up in Customer.io?"**

<!-- para:0008 -->
**User Profile Investigation: ** "Can you look up the profile for user ID 12345 in my Customer.io workspace? I need to see their attributes and engagement history to understand why they're not converting."**

<!-- para:0009 -->
**Workspace Data Exploration: ** "Show me all the custom attributes we're tracking in Customer.io and help me understand which segments we currently have set up for our onboarding flow."**
