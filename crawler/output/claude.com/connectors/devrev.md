<!-- para:0007 -->
**Issue Management: ‍**"Create a new bug report for the login page timeout issue"**

<!-- para:0008 -->
**Ticket Search: ‍**"Find all open support tickets from enterprise customers this week"**

<!-- para:0009 -->
**Knowledge Search: ‍**"Search our knowledge base for documentation about API rate limits"**

<!-- para:0010 -->
**Enhancement Tracking: ‍**"List all enhancements requested for the mobile app and their status"**
