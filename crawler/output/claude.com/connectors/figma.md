# Figma

<!-- para:0002 -->
Generate diagrams and better code from Figma context

<!-- para:0003 -->
- Category: Design
- Used in: [Claude](https://claude.ai/directory/c758d038-d8eb-4421-b426-9dd68dc7f84a) [Claude desktop app](https://claude.com/connectors/figma#) [Claude mobile app](https://claude.com/connectors/figma#) [Claude Code](https://claude.com/connectors/figma#) [Claude Code](https://claude.com/connectors/figma#) [Claude API](https://claude.com/connectors/figma#) [Claude API](https://claude.com/connectors/figma#)

<!-- para:0005 -->
Pull in Figma context and generate high-quality code that aligns with your codebase and design intent. Use the connector to retrieve code resources from Figma Design or Make files, turn your ideas into production apps, and visualize flows as diagrams in Fig Jam.

<!-- para:0006 -->
You can use Figma in Claude to: ‍ Map your flows with diagrams: Turn your prompts into flow charts, Gantt charts, or diagrams in Fig Jam Implement designs as websites: "Help me implement my Figma design as a website in HTML and CSS"

<!-- para:0007 -->
Generate code from selections with annotations: "Get code from the Figma selection and make sure to take into account my annotations"

<!-- para:0008 -->
Check variables for issues: "Get the variables used in my Figma selection, and tell me if there are any issues"

<!-- para:0009 -->
Implement components following codebase standards: "Help me fetch the navbar behavior from my Figma Make file and implement it following my codebase standards"

<!-- para:0010 -->
Connects Figma design components to code components using Code Connect.

<!-- para:0011 -->
Generates custom design system rules for the user's codebase.

<!-- para:0012 -->
Translates Figma designs into production-ready code with 1:1 visual fidelity.

<!-- para:0013 -->
Access Figma skills in the Claude Code Plugin Directory.
