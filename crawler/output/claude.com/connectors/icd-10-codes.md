The ICD-10 Codes connector gives Claude access to the complete ICD-10-CM (diagnosis) and ICD-10-PCS (procedure) code sets for medical classification and billing.

<!-- para:0006 -->
You can use the ICD-10 Codes connector to:

<!-- para:0007 -->
Find diagnosis codes: "I need the ICD-10 code for type 2 diabetes with kidney complications"

<!-- para:0008 -->
Validate codes for claim submission: "I'm submitting a claim with code E11. Is this valid?"

<!-- para:0009 -->
Explore code hierarchies: "Show me how diabetes codes are organized"

<!-- para:0010 -->
Search procedure codes: "Find ICD-10-PCS codes for laparoscopic appendectomy"
