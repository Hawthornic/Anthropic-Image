<!-- para:0005 -->
The Microsoft Learn MCP Server enables clients like Git Hub Copilot and other AI agents to bring trusted and up-to-date information directly from Microsoft's official documentation. It is a remote MCP server that uses streamable http. It allows to search through documentation, fetch a complete article, and search through code samples.

<!-- para:0006 -->
You can use Microsoft Learn to:

<!-- para:0007 -->
**Quick Search & Reference: ** "Give me the Azure CLI commands to create an Azure Container App with a managed identity. search Microsoft Learn"**

<!-- para:0008 -->
**Code Verification & Best Practices: ** "Are you sure this is the right way to implement IHttp Client Factory in a .NET 8 minimal API? search Microsoft Learn and fetch full doc"**

<!-- para:0009 -->
**Comprehensive Learning & Deep Dive: ** "I need to understand Azure Functions end-to-end. search Microsoft Learn and deep dive"**

<!-- para:0010 -->
**Code Sample Search: ** "Show me detailed, runnable python code sample to do harms eval using azure ai foundry evaluation sdk"**
