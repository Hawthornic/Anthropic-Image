# Port IO

<!-- para:0002 -->
Build and query your developer portal and trigger developer workflows

- Used in: [Claude](https://claude.ai/directory/8f5edd1c-c876-465e-a5b9-cc8e6d27dcb7) [Claude desktop app](https://claude.com/connectors/port-io#) [Claude mobile app](https://claude.com/connectors/port-io#) [Claude Code](https://claude.com/connectors/port-io#) [Claude Code](https://claude.com/connectors/port-io#) [Claude API](https://claude.com/connectors/port-io#) [Claude API](https://claude.com/connectors/port-io#)

<!-- para:0005 -->
Connect Port's context lake and software catalog directly to Claude. Query services, dependencies, and ownership using natural language. Analyze scorecards and service health. Execute governed self-service actions and workflows with built-in guardrails.

<!-- para:0006 -->
You can use Port IO to:

<!-- para:0007 -->
**Catalog Search: ** "List all microservices owned by the payments team and their dependencies"**

<!-- para:0008 -->
**Scorecard Analysis: ** "Show me the production readiness scorecard for the checkout service"**

<!-- para:0009 -->
**Self-Service Actions: ** "Run the action to provision a new development environment for my team"**

<!-- para:0010 -->
**Build your developer portal: ** "Create an automation that triggers an AI agent to nudge reviewers through Slack on open pull requests"**
