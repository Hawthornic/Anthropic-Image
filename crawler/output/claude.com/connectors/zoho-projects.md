Control Zoho Projects data and task operations through MCP tools.

<!-- para:0006 -->
You can use Zoho Projects to:

<!-- para:0007 -->
**Create and manage tasks: ** "Create a task called API Integration Review in Project Alpha and assign it to Rahul"**

<!-- para:0008 -->
**Track project progress: ** "List all overdue tasks in Project Alpha with their owners and priority"**

<!-- para:0009 -->
**Manage project phases: ** "Update Project Alpha status to In Progress and add milestone for Q2 delivery"**

<!-- para:0010 -->
**Monitor team workload: ** "Show me all active projects and their current task counts by assignee"**
