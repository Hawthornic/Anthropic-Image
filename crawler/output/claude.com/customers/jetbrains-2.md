# Jet Brains Pioneers Multi-Agent IDE Experience with Claude Agent

<!-- para:0002 -->
*Millions of developers now benefit from the Claude Agent SDK, the same underlying technology that powers Claude Code, natively inside of Jet Brains*

<!-- para:0003 -->
Jet Brains, the company behind IntelliJ IDEA, Py Charm, and other IDEs trusted by 15+ million developers worldwide, has integrated Claude Agent as their first third-party coding agent available directly inside the IDE—no additional plugins or subscriptions required.

<!-- para:0004 -->
Using Anthropic's Claude Agent SDK—the same technology that powers Claude Code—Jet Brains is pioneering an open, multi-agent approach to developer tooling. Claude Agent gives developers frontier agentic coding capabilities within the workflows they already know and love

<!-- para:0005 -->
## The Strategy Move Toward Multi-Agent Choice

<!-- para:0006 -->
Since launching Junie, their first-party coding agent powered by Claude in April 2025, Jet Brains observed strong developer demand for an open AI ecosystem. Internal benchmarks consistently showed Claude's superior performance in code generation, documentation, and refactoring tasks. As Claude Code gained traction as a leading standalone agent, the path forward became clear.

<!-- para:0007 -->
"Claude Code has earned broad recognition among professional developers," said Denis Shiryaev, Group Product Manager at Jet Brains. "We’re excited to build off Anthropic’s Agent SDK to power our Claude Agent, and to make this natively available to our developers. Developers can now use the power of Claude’s agentic capabilities, the same ones that make Claude Code so effective, to accomplish high-value tasks directly inside Jetbrains. This brings the power of Jet Brains and Claude to the next level together.”

<!-- para:0008 -->
The integration aligns with Jet Brains' philosophy of leveraging the best innovations, whether developed internally or with Anthropic, to ensure developers have access to best-in-class tools and the power of choice.

<!-- para:0009 -->
## Seamless Integration

<!-- para:0010 -->
Claude Agent is included in the Jet Brains AI subscription, at no extra cost, alongside Junie. This unified subscription model eliminates complexity while giving developers flexibility with which agent to use in their work.

<!-- para:0011 -->
Key advantages:

<!-- para:0012 -->
- **Native IDE access: **Claude Agent operates directly within Jet Brains' AI Chat interface, accessing IDE functionality through the Jet Brains MCP server. Developers can navigate codebases, execute searches, and apply refactorings without switching contexts
- **No deployment overhead: ** No separate plugins to install, authentication flows, or standalone tools to manage
- **Unified experience: **All AI assistance appears in a single chat interface via Jet Brains AI Assistant, streamlining workflows

<!-- para:0013 -->
## An Open Ecosystem Vision

<!-- para:0014 -->
The Claude Agent integration represents Jet Brains' first step toward an open AI ecosystem within their IDEs—giving developers real choice.

<!-- para:0015 -->
"This is just the beginning of opening up our AI ecosystem," Melnikov explained. "Developers will have the freedom to work with various agents, all seamlessly integrated into their IDE. No extra plugins to install. No extra subscriptions to manage."

<!-- para:0016 -->
This approach acknowledges that different developers have different preferences and different tasks may call for different AI capabilities. By supporting multiple agents within a consistent interface, Jet Brains lets developers optimize their workflows without sacrificing the integrated experience their IDEs are known for.

<!-- para:0017 -->
## **Build on Claude Code’s Foundation**

<!-- para:0018 -->
By building atop the Claude Agent SDK, Jet Brains leverages the same foundational building blocks that make Claude Code effective– context management, file operations, tool calls and code execution–-all now native within the Jet Brains IDEs that developers love.

<!-- para:0019 -->
Jet Brain's launch of Claude Agent represents more than a product integration. It demonstrates how Claude Code's success can extend into open, multi-agent developer ecosystems. As Jet Brains continues expanding their AI ecosystem, they’re taking a leadership role in defining the future of AI-powered software development.
