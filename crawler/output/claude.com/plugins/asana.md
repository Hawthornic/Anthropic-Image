# Asana

<!-- para:0002 -->
Asana integration: Create and manage tasks, search projects, update assignments, track progress in your development workflow.

<!-- para:0003 -->
- Install in [Claude Code](https://claude.com/plugins/asana#)
- Made by: [Asana](https://asana.com/)
- Installs6317

<!-- para:0004 -->
Connect Claude Code to Asana's work management platform with 30+ powerful tools for project and task management. Create tasks, track project status, manage assignments, search across workspaces, and update goals—all through natural language commands directly in your development environment.

<!-- para:0005 -->
Key capabilities include task creation and management, project tracking and status updates, user and team organization, goal updates, typeahead search, and section management. The integration uses OAuth authentication with your Asana account for secure access to your workspace data.

<!-- para:0006 -->
**How to use: ** After connecting your Asana account, interact naturally with your workspace. Try prompts like "Find all my incomplete tasks due this week", "Create a new task in the Marketing project assigned to me", "List all sections in the Product Launch project", or "Show me the status of the Q2 Planning project".**
