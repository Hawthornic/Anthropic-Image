# Bio Research

<!-- para:0002 -->
Connect to preclinical research tools and databases to accelerate early-stage life sciences R&D.

<!-- para:0003 -->
- Anthropic Verified
- Install in [Claude Cowork](https://claude.ai/desktop/customize/plugins/new?marketplace=anthropics/knowledge-work-plugins&plugin=bio-research)
- Made by: [Anthropic](https://anthropic.com/)
- Installs0

<!-- para:0004 -->
Bio Research consolidates 10 MCP server integrations and 6 analysis skills into a single package for life science researchers. Search biomedical literature via Pub Med and preprint servers, access journal articles, explore drug target databases, and browse clinical trial registries — all from one plugin.

<!-- para:0005 -->
The plugin includes deep analysis workflows: run automated quality control on single-cell RNA-seq data, use scvi-tools for batch correction and cell type annotation, or launch nf-core bioinformatics pipelines for RNA-seq, variant calling, and ATAC-seq on local or public GEO/SRA data.

<!-- para:0006 -->
For translational work, generate FDA/NIH-compliant clinical trial protocols, convert lab instrument output files to Allotrope Simple Model format (supporting 40+ instrument types), and apply a systematic research problem selection framework.

<!-- para:0007 -->
Common workflows include literature review, single-cell analysis, sequencing pipelines, drug discovery research, and clinical trial design. Use commands like `/start` to see available tools and get oriented.
