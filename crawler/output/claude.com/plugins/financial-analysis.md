# Financial Analysis

<!-- para:0002 -->
Build DCF models, run comps analysis, structure LBOs, and review pitch decks

<!-- para:0003 -->
- Anthropic Verified
- Install in [Claude Cowork](https://claude.ai/desktop/customize/plugins/new?marketplace=anthropics/financial-services-plugins&plugin=financial-analysis)
- Made by: [Anthropic](https://anthropic.com/)
- Installs

<!-- para:0004 -->
Build DCF models, run comparable company analysis, structure LBOs, and review financial models and pitch decks.

<!-- para:0005 -->
You can use Financial Analysis to:

<!-- para:0006 -->
**DCF Model: ** "Build a discounted cash flow model for Tesla with a 5-year projection"**

<!-- para:0007 -->
**Comps Analysis: ** "Run a comparable company analysis for Stripe against public fintech peers"**

<!-- para:0008 -->
**LBO Model: ** "Structure an LBO for a $2B enterprise software company at 6x EBITDA"**

<!-- para:0009 -->
**Deck Review: ** "Review this pitch deck for errors, inconsistencies, and missing data"**
