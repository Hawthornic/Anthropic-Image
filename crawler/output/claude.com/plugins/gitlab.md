# Git Lab

<!-- para:0002 -->
Git Lab integration for repositories, merge requests, CI/CD pipelines, issues, and wikis with full Dev Ops lifecycle access.

<!-- para:0003 -->
- Install in [Claude Code](https://claude.com/plugins/gitlab#)
- Made by: [Git Lab](https://gitlab.com/)
- Installs20491

<!-- para:0004 -->
Connect Claude Code directly to your Git Lab instance for seamless Dev Ops integration. This plugin provides full access to Git Lab's comprehensive platform, enabling you to manage repositories, work with merge requests, monitor CI/CD pipelines, track issues, and edit wiki documentation—all from within your Claude Code workflow.

<!-- para:0005 -->
Key capabilities include browsing and managing Git repositories, creating and reviewing merge requests, viewing pipeline status and job logs, managing project issues and milestones, and working with project wikis. The integration uses Git Lab's official MCP API for reliable, native access to the platform.

<!-- para:0006 -->
**How to use: ** Once installed and authenticated with your Git Lab account, you can interact with your Git Lab projects naturally. Try prompts like "show me open merge requests in my project," "what's the status of the latest pipeline," "create an issue for the authentication bug," or "list recent commits on the main branch." The plugin works with both Git Lab.com and self-managed Git Lab instances.**
