# Learning Output Style

<!-- para:0002 -->
Interactive learning mode requesting meaningful code contributions at key decision points

<!-- para:0003 -->
- Anthropic Verified
- Install in [Claude Code](https://claude.com/plugins/learning-output-style#)
- Made by: [Anthropic](https://anthropic.com/)
- Installs25495

<!-- para:0004 -->
Transform your Claude Code sessions into interactive learning experiences. This plugin activates a learning mode that pauses at meaningful decision points and requests 5-10 lines of code from you, rather than providing complete solutions. You'll develop hands-on skills by implementing business logic, error handling strategies, and architectural decisions yourself.

<!-- para:0005 -->
The plugin intelligently distinguishes between code worth writing yourself (algorithm choices, design patterns, data structures) and boilerplate Claude should handle directly (configuration, CRUD operations, obvious implementations). When requesting your contribution, Claude sets up the file context with function signatures and explains the trade-offs involved.

<!-- para:0006 -->
Additionally, the plugin provides educational insights during coding sessions—brief explanations about codebase patterns, design decisions, and implementation approaches formatted in distinct insight blocks.

<!-- para:0007 -->
**How to use: ** Once installed, the plugin activates automatically at session start. Simply work on your coding tasks as usual, and Claude will naturally pause at strategic points to request your input. No slash commands or special invocations required—just start coding and learn by doing.**
