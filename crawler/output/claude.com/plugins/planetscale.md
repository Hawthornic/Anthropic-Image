# Planet Scale

<!-- para:0002 -->
An authenticated hosted MCP server that accesses your Planet Scale organizations, databases, branches, schema, and Ins...

<!-- para:0003 -->
- Install in [Claude Code](https://claude.com/plugins/planetscale#)
- Made by: [Planet Scale](https://planetscale.com/)
- Installs509

<!-- para:0004 -->
Connect Claude directly to your Planet Scale databases with an authenticated MCP server. Browse your organizations, databases, branches, and schema. Run queries against your data, surface slow queries through Planet Scale Insights, and get organizational and account information — all from within Claude Code.

<!-- para:0005 -->
The plugin also includes expert database skills for MySQL, PostgreSQL, Vitess, and Planet Scale's sharded Postgres offering (Neki). These skills give Claude deep knowledge of schema design, indexing strategies, query optimization, transaction handling, sharding patterns, VSchema setup, online DDL operations, and production-safe migration workflows.

<!-- para:0006 -->
**How to use: ** After installing, Claude can access your Planet Scale resources through natural language. Try prompts like:**

<!-- para:0007 -->
- "List all databases in my Planet Scale organization"
- "Show the schema for the users table on the main branch"
- "Find the slowest queries hitting my production database"
- "Help me design an index strategy for this query pattern"
- "What's the best way to run this schema migration safely on Planet Scale?"
