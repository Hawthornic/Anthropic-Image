# Postiz

<!-- para:0002 -->
Social media automation CLI: schedule posts, manage integrations, upload media, track analytics (28+ platforms)

<!-- para:0003 -->
- Install in [Claude Code](https://claude.com/plugins/postiz#)
- Made by: [Nevo David](https://github.com/nevo-david)
- Installs248

<!-- para:0004 -->
Postiz is a social media automation plugin that lets you schedule posts, manage integrations, upload media, and track analytics across 28+ platforms — including X, Linked In, Reddit, You Tube, Tik Tok, Instagram, Facebook, and more — all from within Claude Code. It provides a full suite of CLI commands for discovering connected accounts, creating multi-platform campaigns, and monitoring post performance.

<!-- para:0005 -->
Key features include multi-platform posting with a single command, threaded posts with follow-up comments and per-comment media, scheduled publishing with precise timestamps, platform-specific settings discovery (e.g. Reddit flairs, You Tube playlists), media uploads (images, videos, audio, documents), and analytics tracking for both platform-level and individual post engagement.

<!-- para:0006 -->
**How to use: ** Set your `POSTIZ_API_KEY` environment variable, then try prompts like:**

<!-- para:0007 -->
"List all my connected social media integrations" · "Schedule a post to Linked In and X for tomorrow at 9am announcing our new feature" · "Upload this image and create a post with it on Instagram" · "Show me analytics for my recent posts" · "Create a multi-platform campaign with a main post and follow-up comments" · "What Reddit flairs are available for my connected subreddit?"

<!-- para:0008 -->
The plugin supports both simple single-post workflows and advanced JSON-based campaigns with custom delays, draft mode, URL shortening, tags, and batch operations — making it well-suited for automated social media management at scale.
