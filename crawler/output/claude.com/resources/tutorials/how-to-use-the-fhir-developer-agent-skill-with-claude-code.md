## **What is the FHIR Developer agent skill for Claude Code?**

<!-- para:0004 -->
FHIR agent skill helps Claude Code connect healthcare systems faster and with fewer errors, improving the interoperability of medical data. The FHIR skill provides Claude Code with specialized knowledge of the HL7 FHIR R4 standard for healthcare data exchange, including resource structures, required vs optional fields (cardinality rules), coding systems (LOINC, SNOMED CT, Rx Norm), RESTful API patterns, and validation implementation patterns

<!-- para:0005 -->
## **Who should use the FHIR Developer agent skill?**

<!-- para:0006 -->
This agent skill is designed for healthcare software developers and integrators who need to build FHIR-compliant REST APIs, convert healthcare data to FHIR format, or implement clinical data validation without writing code from scratch.
